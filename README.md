PDFちょき 1.1

PDFちょき 1.1 は、
Google 翻訳で PDF を扱いやすくするための補助アプリです。

大きな PDF ファイルを、
1ファイルあたり 10MB 以下になるように分割します。

できること

PDF ファイルを 10MB 以下の PDF に分割します

Google 翻訳にアップロードできるサイズに調整します

ローカル環境（ブラウザ）だけで動作します

ファイルを サーバーへ送信しません

分割ルール（PDFちょき 1.1）

24ページを目安に分割します

分割後のファイルサイズが 10MB を超えた場合

ページ数を 3ページずつ減らして再計測します

3ページでも 10MB を超える場合

その PDF は処理できません（エラー表示）

※ 高精細な PDF では、ページ数が少なくても制限に達する場合があります。

使い方

分割したい PDF ファイルを選択

保存先フォルダを指定

「スタート」 を押す

分割された PDF が保存されます

想定用途

Google 翻訳で PDF を翻訳するための下準備

大きな PDF を分割して扱いやすくする

注意事項

1ページだけで 10MB を超える高精細 PDF は処理できません

本アプリは 翻訳そのものを行うアプリではありません

PDF の内容は 一切変更されません（分割のみ）

バージョン

PDFちょき 1.1
