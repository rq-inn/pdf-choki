# ✂️ PDF ちょき 1.0

PDFファイルを **10MB以下・ページ単位** に安全に分割する、  
シンプルで軽快なブラウザ完結型ツールです。

インストール不要。  
**通信なし・ローカル完結**で動作します。

---

## 🔹 特徴

- 📄 PDFをページ単位で自動分割
- 📦 各ファイルは **10MB以下** に調整
- 🔒 ファイルは外部に送信されません（ローカル処理）
- 🖥 Windows / macOS / Linux / Chrome / Edge 対応
- 📱 PWA対応（ホーム画面に追加可能）

---

## 🔹 使い方

1. **「読込ファイルを選択してください」**  
   → 分割したいPDFを選択

2. **「保存先フォルダを指定してください」**  
   → 分割後PDFの保存先を指定

3. **「スタート」** をクリック  
   → 自動で分割・保存されます

進捗は下部のバーで確認できます。

---

## 🔹 動作仕様

- 1ファイル最大：**10MB**
- 最大ページ数：30ページ（自動調整）
- 奇数ページが残らないよう調整
- 分割不可の場合は処理を中断

---

## 🔹 対応環境

- Google Chrome
- Microsoft Edge
- Chromium系ブラウザ
- Android（PWA）
- Windows / macOS / Linux

※ Safari では動作しません。
