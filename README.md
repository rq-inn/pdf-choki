PDFちょき 1.0

PDF ファイルを 30MB 以下に分割する、ブラウザ完結型ツールです。
インストール不要・ローカル処理のみで動作します。

対応環境

Windows / macOS（PC ブラウザ）

Chrome / Edge / Firefox 推奨

iOS / Android

非対応（注意メッセージを表示）

※ iOS / Android では、ブラウザの仕様により
PDF 分割に必要なファイル操作が制限されるため対応していません。

主な機能

PDF ファイルを 30MB 以下で自動分割

最大 50 ページまでを 1 ファイルとしてまとめる

分割結果は連番で保存

進捗バー表示

多言語対応（日本語 / English / 繁体字）

分割ルール（重要）

分割後の PDF は 1 ファイルあたり最大 30MB

1 チャンクの最大ページ数は 50 ページ

偶数ページを優先して分割

1ページの容量が 10MB を超える PDF を含む場合は処理不可

その場合、以下のエラーメッセージが表示されます：

大容量ページを含んでいます。このPDFファイルは扱えません。

使い方

「読込ファイルを選択してください」
分割したい PDF を選択

「保存先フォルダを指定してください」
出力先フォルダを指定

「スタート」 をクリック
分割処理が開始されます

注意事項

処理はすべて ローカル（ブラウザ内） で行われます

PDF ファイルは外部に送信されません

大きな PDF の場合、処理に時間がかかることがあります
